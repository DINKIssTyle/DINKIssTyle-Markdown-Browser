# Shortcuts

## App

- `Cmd/Ctrl + S`: Save current document
- `Cmd/Ctrl + W`: Close current tab
- `Cmd/Ctrl + R`: Refresh current document
- `Cmd/Ctrl + Shift + H`: Go to Start page
- `Cmd/Ctrl + 1-9`: Switch to tab 1-9

## Navigation

- `Cmd/Ctrl + [`: Back
- `Cmd/Ctrl + ]`: Forward

## Editor

- `Cmd/Ctrl + E`: Enter or exit edit mode
- `Cmd/Ctrl + -`: Decrease editor font size when the editor has focus
- `Cmd/Ctrl + +`: Increase editor font size when the editor has focus

## Prompt

- Select text in the editor to open the AI prompt
- Press `/` in the AI prompt to type an instruction quickly
