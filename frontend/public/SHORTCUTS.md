# ⌨️ Shortcuts
This is the list of shortcuts for the DKST Markdown Browser.

## Common Shortcuts
| Action | Windows/Linux | macOS |
| :--- | :--- | :--- |
| Open New Tab | Ctrl+T | Cmd+T |
| Close Current Tab | Ctrl+W | Cmd+W |
| Go to Tab | Ctrl+Number | Cmd+Number |

## Reader Mode
| Action | Windows/Linux | macOS |
|---|---|---|
| Open File | Ctrl+O | Cmd+O |
| Go to Home | Ctrl+Shift+H | Cmd+Shift+H |
| Refresh | Ctrl+R | Cmd+R |
| Search | Ctrl+F | Cmd+F |
| Change Theme | Ctrl+T | Cmd+T |
| History Navigation | Ctrl+[ , Ctrl+] | Cmd+[ , Cmd+] |
| Font Size | Ctrl++, Ctrl+- | Cmd++, Cmd+- |

## Edit Mode
**💡 Tip:** Switching between Leader Mode and Editor Mode is very fast using the shortcut **`Ctrl + E`** (Windows/Linux) or **`Cmd + E`** (macOS)!
| Feature | Shortcut/Usage | Description |
|---|---|---|
| Search Code Shortcuts | `/` + Arrow Key Enter or type feature name Enter | Searches all available code shortcuts on the keyboard. |
| Ask AI | Select text, then press `/` key | Receives help from AI regarding the selected text. |
| General text | Follow OS default shortcuts | General text editing functions follow the OS default shortcuts. |
